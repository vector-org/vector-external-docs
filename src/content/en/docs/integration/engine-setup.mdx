# Engine Setup

Welcome to the Vector DRM integration guide for game engines. Currently, **only Unity and Unreal Engine are supported** for DRM integration via the Vector SDK.

## Supported Engines & Requirements

### Unity
- **Supported Version:** Unity 2021.3 LTS or newer (recommended: latest LTS)
- **Download:** [Unity Hub & Editor](https://unity.com/download)

### Unreal Engine
- **Supported Version:** Unreal Engine 5.0 or newer
- **Download:** [Unreal Engine](https://www.unrealengine.com/en-US/download)

> ⚠️ *Other engines are not supported at this time. Please ensure you are using a compatible version before proceeding.*
