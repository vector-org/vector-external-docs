# Installing Vector Client

The Vector Client is required for both gamers and developers to interact with the Vector ecosystem. It manages DRM tokens, user authentication, and game entitlement.

## Downloading Vector Client

1. Visit the official download portal.
2. Select your operating system and machine architecture.
3. Download the latest compatible version of the Vector Client.
