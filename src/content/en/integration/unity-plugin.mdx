## VectorSDK Plugin for Unity

**Version:** 0.1.0  
**Author:** Vector  
**Contact:** security@playonvector.com  
**License:** VECTOR Commercial Use License

---

### Overview

The VectorSDK Unity plugin provides secure, hardware-locked DRM for your game using a cross-platform Rust shared library. At startup, it validates a developer-provided Game UUID and a user-local DRM token (`drm-token.json`, auto-generated by vector-client), enforcing license checks before gameplay.

> **Note:** The `drm-token.json` file is automatically generated by installing and running the `vector-client` application. You do not need to create or edit this file manually. Ensure the client is installed and started on the user's machine before launching the game.

**Supported Platforms:**  
- Windows (x64)  
- macOS (Apple Silicon)  
- Linux (x64)

---

### Key Features

- Hardware-locked DRM token validation at startup
- Secure shutdown if validation fails
- Cross-platform support
- Game UUID assignment via Unity Inspector
- Lightweight, zero runtime dependencies
- Native performance (Rust shared library)

---

### Integration Guide

**Step 1: Add the Plugin**  
Place the `VectorSDK.cs` script and binaries in your project:

```
YourUnityProject/
└── Assets/
  └── Plugins/
    └── VectorSDK/
      ├── VectorSDK.cs
      └── Binaries/
        ├── Windows/vector_sdk.dll
        ├── Mac/libvector_sdk.dylib
        └── Linux/libvector_sdk.so
```

#### Plugin Script Example

```csharp
using System;
using System.Runtime.InteropServices;
using System.IO;
using UnityEngine;

public class VectorSDK : MonoBehaviour
{
#if UNITY_EDITOR_WIN || UNITY_STANDALONE_WIN
  private const string LIB_NAME = "vector_sdk.dll";
#elif UNITY_EDITOR_OSX || UNITY_STANDALONE_OSX
  private const string LIB_NAME = "libvector_sdk.dylib";
#elif UNITY_EDITOR_LINUX || UNITY_STANDALONE_LINUX
  private const string LIB_NAME = "libvector_sdk.so";
#else
  private const string LIB_NAME = "__internal_error_unsupported_platform__";
#endif

  private const string PLUGIN_VERSION = "0.1.0";

  [SerializeField] private string gameUuid;

  [DllImport(LIB_NAME, CallingConvention = CallingConvention.Cdecl)]
  private static extern IntPtr load_and_validate_drm_token(IntPtr drm_token_path, IntPtr game_uuid);
  [DllImport(LIB_NAME, CallingConvention = CallingConvention.Cdecl)]
  private static extern IntPtr get_vector_sdk_version();

  [DllImport(LIB_NAME, CallingConvention = CallingConvention.Cdecl)]
  private static extern void free_string(IntPtr ptr);

  void Start()
  {
    Debug.Log("⚙️ Initializing VectorSDK plugin...");

    if (string.IsNullOrEmpty(gameUuid))
    {
      Debug.LogError("❌ Game UUID is empty. Please assign it in the inspector.");
#if UNITY_EDITOR
      UnityEditor.EditorApplication.isPlaying = false;
#else
      Application.Quit();
#endif
      return;
    }

    string libVersion = GetLibraryVersion();
    if (!IsLibraryVersionCompatible(libVersion, PLUGIN_VERSION))
    {
      Debug.LogError($"VectorSDK native library is outdated (v{libVersion}). Please update your Vector Client to use plugin v{PLUGIN_VERSION}.");
#if UNITY_EDITOR
      UnityEditor.EditorApplication.isPlaying = false;
#else
      Application.Quit();
#endif
      return;
    }

    try
    {
      string tokenPath = VectorSDKPathHelper.GetDRMTokenPath();

      if (!File.Exists(tokenPath))
      {
        Debug.LogError($"❌ Token file not found at path: {tokenPath}");
#if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
#else
        Application.Quit();
#endif
        return;
      }

      string result = ValidateToken(tokenPath, gameUuid);

      if (result.Contains("error", StringComparison.OrdinalIgnoreCase) ||
        result.Contains("invalid", StringComparison.OrdinalIgnoreCase) ||
        result.Contains("expired", StringComparison.OrdinalIgnoreCase) ||
        result.Contains("failed", StringComparison.OrdinalIgnoreCase))
      {
        Debug.LogError($"❌ Token validation failed: {result}");
#if UNITY_EDITOR
        UnityEditor.EditorApplication.isPlaying = false;
#else
        Application.Quit();
#endif
      }
      else
      {
        Debug.Log($"✅ Token validation successful: {result}");
      }
    }
    catch (Exception ex)
    {
      Debug.LogError($"❌ VectorSDK initialization failed: {ex.Message}");
#if UNITY_EDITOR
      UnityEditor.EditorApplication.isPlaying = false;
#else
      Application.Quit();
#endif
    }
  }

  public static string ValidateToken(string drmTokenPath, string gameUuid)
  {
    if (string.IsNullOrEmpty(drmTokenPath) || string.IsNullOrEmpty(gameUuid))
    {
      Debug.LogError("❌ Token path or UUID is null or empty.");
      return "Error: Invalid input parameters";
    }

    IntPtr pathPtr = IntPtr.Zero;
    IntPtr uuidPtr = IntPtr.Zero;

    try
    {
      pathPtr = Marshal.StringToHGlobalAnsi(drmTokenPath);
      uuidPtr = Marshal.StringToHGlobalAnsi(gameUuid);

      IntPtr resultPtr = load_and_validate_drm_token(pathPtr, uuidPtr);
      if (resultPtr == IntPtr.Zero)
      {
        return "Error: Validation returned null";
      }

      string result = Marshal.PtrToStringAnsi(resultPtr);
      free_string(resultPtr);

      return result;
    }
    catch (DllNotFoundException ex)
    {
      Debug.LogError($"❌ VectorSDK library not found: {ex.Message}");
      return $"Error: Library not found - {ex.Message}";
    }
    catch (EntryPointNotFoundException ex)
    {
      Debug.LogError($"❌ VectorSDK function not found: {ex.Message}");
      return $"Error: Function not found - {ex.Message}";
    }
    catch (Exception ex)
    {
      Debug.LogError($"❌ Rust validation failed: {ex.Message}");
      return $"Error: {ex.Message}";
    }
    finally
    {
      if (pathPtr != IntPtr.Zero)
        Marshal.FreeHGlobal(pathPtr);
      if (uuidPtr != IntPtr.Zero)
        Marshal.FreeHGlobal(uuidPtr);
    }
  }

  private static string GetLibraryVersion()
  {
    try
    {
      IntPtr versionPtr = get_vector_sdk_version();
      if (versionPtr == IntPtr.Zero)
      {
        Debug.LogError("❌ Library returned null pointer for version");
        return "0.0.0";
      }

      string version = Marshal.PtrToStringAnsi(versionPtr);
      free_string(versionPtr);
      return version;
    }
    catch (DllNotFoundException ex)
    {
      Debug.LogError($"❌ VectorSDK library not found: {ex.Message}");
      return "0.0.0";
    }
    catch (EntryPointNotFoundException ex)
    {
      Debug.LogError($"❌ get_vector_sdk_version function not found: {ex.Message}");
      return "0.0.0";
    }
    catch (Exception ex)
    {
      Debug.LogError($"❌ Failed to read library version: {ex.Message}");
      return "0.0.0";
    }
  }

  private static bool IsLibraryVersionCompatible(string libVersion, string pluginVersion)
  {
    Version lib = new Version(libVersion);
    Version plugin = new Version(pluginVersion);
    return lib >= plugin;
  }
}

public static class VectorSDKPathHelper
{
  public static string GetDRMTokenPath()
  {
    string basePath;

#if UNITY_STANDALONE_OSX || UNITY_EDITOR_OSX
    basePath = Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.Personal), "Library", "Application Support");
#elif UNITY_STANDALONE_WIN || UNITY_EDITOR_WIN
    basePath = Environment.GetFolderPath(Environment.SpecialFolder.ApplicationData);
#elif UNITY_STANDALONE_LINUX || UNITY_EDITOR_LINUX
    basePath = Path.Combine(Environment.GetFolderPath(Environment.SpecialFolder.Personal), ".config");
#else
    basePath = Application.persistentDataPath;
#endif

    return Path.Combine(basePath, "com.vector.client", "drm-token.json");
  }
}
```

**Step 2: Add Platform Binaries**  
- Download the latest binaries from:  
  https://github.com/vector-org/vector-sdk/releases/tag/v0.0.3  
- Place each file in the correct `Binaries/<Platform>/` directory.
- Unity does not copy native binaries automatically; use post-build scripts if needed.
- On macOS, you may need to allow unnotarized binaries manually.

**Step 3: Set Your Game UUID**  
- Add `VectorSDK.cs` to a persistent GameObject (e.g., `GameManager`)
- In the Inspector, assign your Game UUID
- Save the scene  
- If the UUID is missing, the game will terminate at startup

**Step 4: Build and Package**  
- Go to `File > Build Settings`
- Select your platform and build location
- After building, ensure:
  - `vector_sdk.dll` is next to the `.exe` (Windows)
  - `libvector_sdk.dylib` is in `.app/Contents/Plugins/` (macOS)
  - `libvector_sdk.so` is beside the game binary or in a `Plugins/` directory (Linux)
- Manually copy binaries as needed

---

### Validation Behavior

- **Success:** Game launches normally
- **Failure:** Play mode stops or app exits, with a log message (e.g., bad token, missing UUID, missing binary)

---

### Troubleshooting

- **Game exits on startup:** Check Unity Console or Player logs. Ensure Game UUID is set, correct binary is present, and token file exists.
- **DllNotFoundException/EntryPointNotFoundException:** Binary missing, wrong path, or permissions issue.
- **DRM token location:**  
  | Platform | Token Path |
  |----------|------------|
  | Windows  | `%APPDATA%\\com.vector.client\\drm-token.json` |
  | macOS    | `~/Library/Application Support/com.vector.client/drm-token.json` |
  | Linux    | `~/.config/com.vector.client/drm-token.json` |
