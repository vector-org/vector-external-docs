### Overview

The VectorSDK Unreal plugin enables secure, hardware-locked DRM for your game using a cross-platform Rust shared library. At startup, it validates a developer-provided Game UUID and a user-local DRM token (`drm-token.json`, auto-generated by vector-client), enforcing license checks before gameplay.

**Supported Platforms:**  
- Windows (x64)  
- macOS (Apple Silicon)  
- Linux (x64)

---

### Key Features

- Hardware-locked DRM token validation at startup
- Secure shutdown if validation fails
- Cross-platform support
- Game UUID injection via config or command line
- Lightweight, zero runtime dependencies
- Native performance (Rust shared library)

### Integration Guide

**Step 1: Add the Plugin**  
Place the `VectorSDK/` directory inside your project's `Plugins/` folder:

```
YourProject/
└── Plugins/
    └── VectorSDK/
        ├── Binaries/
        │   ├── Win64/vector_sdk.dll
        │   ├── Mac/libvector_sdk.dylib
        │   └── Linux/libvector_sdk.so
        ├── Source/VectorSDK/
        │   └── VectorSDK.cpp
        └── VectorSDK.uplugin
```

**Step 2: Add Platform Binaries**  
- Download the latest binaries from:  
  https://github.com/vector-org/vector-sdk/releases/tag/v0.0.3  
- Place each file in the correct `Binaries/<Platform>/` directory.
- Note: Binaries are unsigned; you may need to allow them manually.

**Step 3: Enable the Plugin**  
- Open your Unreal project  
- Go to `Edit > Plugins`  
- Search for "VectorSDK" and enable it  
- Restart the editor

**Step 4: Set Your Game UUID**  
- **Option A:** Add to `Config/DefaultGame.ini`:
  ```
  [/Script/Engine.GameGeneralSettings]
  VectorSDKGameUUID=your-unique-game-uuid-here
  ```
- **Option B:** Pass via command line:
  ```
  -VectorSDKGameUUID=your-unique-game-uuid-here
  ```
- If the UUID is missing, the plugin will terminate the game.

**Step 5: Build and Package**  
- In Unreal Editor: `File > Package Project`
- Select your platform
- Ensure the shared library is included in the final package under `Plugins/VectorSDK/Binaries/<Platform>/`
- For CI/CD, copy the shared library manually if needed

---

### Validation Behavior

- **Success:** Game launches normally
- **Failure:** Error dialog is shown and the game exits (e.g., bad token, missing UUID)

---

### Troubleshooting

- **Game exits on startup:** Check logs, ensure token file and Game UUID are set, and shared library is present.
- **Plugin not visible:** Ensure it's in `Plugins/` and restart the editor.
- **Manual validation:** Launch with `-VectorSDKGameUUID=test-game-uuid`
- **DRM token location:**  
  | Platform | Token Path |
  |----------|------------|
  | Windows  | `%APPDATA%\\com.vector.client\\drm-token.json` |
  | macOS    | `~/Library/Application Support/com.vector.client/drm-token.json` |
  | Linux    | `~/.config/com.vector.client/drm-token.json` |
