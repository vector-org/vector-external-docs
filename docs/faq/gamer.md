
# Gamer FAQ

<details>
<summary>How do I install the Vector client?</summary>
Download the Vector client for your platform from the official portal and follow the installation guide in this documentation.
</details>

<details>
<summary>Where is my DRM token stored?</summary>
The DRM token is stored in your user configuration directory. See the installation guide for platform-specific paths.
</details>

<details>
<summary>What platforms are supported?</summary>
Vector supports Windows (x64), macOS (Apple Silicon), and Linux (x64).
</details>

<details>
<summary>What should I do if a game does not launch?</summary>
Ensure the Vector client is running, your DRM token is valid, and your game files are up to date. See troubleshooting for more help.
</details>
